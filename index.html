Ollama Cloud

1. Go to ollama.com/settings/keys and sign in (or create a free account).
2. Click "New API key", give it a name, and copy it.
3. Paste it here — it's stored only on this device, never sent anywhere except Ollama.
Note: the API Base URL below is just a CORS proxy — your key itself always comes from ollama.com, not from the proxy URL.
Default: https://ollama.com. If you run the proxy.js locally, set this to http://localhost:11435. If you're using a deployed Cloudflare Worker, use its *.workers.dev URL instead.
Check ollama.com/search?c=cloud for current cloud model tags — naming changes over time.

Calorie goal

No single number fits everyone — a common range is ~1,800–2,200 kcal for women and ~2,200–2,800 for men, depending on age, weight and activity. Adjust to what your doctor or dietitian recommends. Tracked independently from your sugar streak.

Sugar streak

Set to 0 if any added sugar at all should break your streak. Otherwise, set a small allowance (e.g. 5g). Going over resets your streak. Tracked independently from your calorie goal — logging calories never affects it.
